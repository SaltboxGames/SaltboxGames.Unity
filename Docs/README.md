# SaltboxGames.Unity Documentation

Documentation for the runtime and editor files in `SaltboxGames.Unity`.

## Runtime Areas

- [Addressables](./addressables.md): optional scene references and SafeGuid address lookup helpers.
- [Extensions](./extensions.md): Unity-specific extension methods.
- [FX](./fx.md): optional camera and freeze-frame effects.
- [Services](./services.md): Unity component wrappers for SaltboxGames.Core services.
- [Type References](./type-references.md): serializable type references and editor selection support.
- [Utilities](./utilities.md): player-loop, logging, and inspector utility helpers.

## Editor Areas

- [Editor Utilities](./editor.md): editor-only reflection and serialized property helpers.

## Installation

`SaltboxGames.Unity` requires `SaltboxGames.Core`. See the package [README](../README.md) for Unity installation notes, dependency setup, and optional integration details.
